I am a multidisciplinary designer and developer based in Tokyo. For the last decade, I have been obsessed with the web as a medium for thought—not just a platform for applications.

This site is my Digital Garden. Unlike a blog, which is a chronological stream, a garden is a collection of evolving notes. Some are seedlings (just started), some are evergreen (mature thoughts).

I believe in software that respects the user's attention, interfaces that feel like physical objects, and code that lasts.
